### Mục tiêu tuần 4:

* Tìm hiểu Amazon EC2 và cách triển khai một web server cơ bản.

### Kết quả đạt được tuần 4:

* Hiểu Amazon EC2 là dịch vụ dùng để tạo và quản lý máy chủ ảo trên AWS.
* Nắm được các thành phần cơ bản khi tạo EC2 như AMI, instance type, key pair, EBS và security group.
* Tạo được EC2 instance phục vụ mục đích triển khai web server.
* Cài đặt được Nginx và truy cập được web server thông qua public IP.
* Deploy được một website tĩnh cơ bản lên EC2 sử dụng Nginx.
