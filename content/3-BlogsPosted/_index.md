This section will list and introduce the blogs you have posted to [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). For example:

###  [Blog 1 - BUILDING AN OFFLINE-FIRST APPLICATION WITH AWS AMPLIFY, TANSTACK QUERY, APPSYNC AND MONGODB ATLAS](3.1-Blog1/)
This blog introduces how to build an Offline-First application combined with Optimistic UI using AWS Amplify Gen 2, AWS AppSync, TanStack Query, and MongoDB Atlas, keeping the app responsive while still syncing data correctly once the network connection is restored.

###  [Blog 2 - EMBEDDING A GENAI ASSISTANT INTO AN INTERNAL APPLICATION WITH AWS AMPLIFY, CDK AND AMAZON Q BUSINESS](3.2-Blog2/)
This blog explains how to embed a document-grounded AI assistant into a web application using Amazon Q Business, combined with AWS Amplify Gen 2 and AWS CDK to manage infrastructure, authentication, and data indexing from Amazon S3.

###  [Blog 3 - INTEGRATING IAM COMPUTE ROLES FOR SERVER-SIDE RENDERING APPS ON AWS AMPLIFY HOSTING](3.3-Blog3/)
This blog introduces IAM Compute Roles on AWS Amplify Hosting, which let SSR applications (Next.js, Nuxt, etc.) securely access AWS services such as S3, RDS, DynamoDB, and Secrets Manager through temporary credentials without storing Access Keys or Secret Keys in source code.
